# 阶段三：稳定 — 阶段概述

**时间**：2026-08-30 ~ 2026-10-10（第 11-16 周，计划）
**目标**：单元测试补全、异常场景覆盖、性能优化、用户操作完善、打包发布

---

## 各周计划

### 第 11 周：单元测试补全
- ImageLib 算子测试（覆盖率目标 70%+）
- WorkflowEngine 测试（覆盖率目标 80%+）
- CommLib 适配器测试（覆盖率目标 60%+）
- SystemLib 测试（覆盖率目标 70%+）
- CI 流水线配置（GitHub Actions，每次提交自动运行测试）
- 覆盖率报告生成 + 未覆盖分支分析

### 第 12 周：异常场景覆盖
- 相机断线 → 自动重连（重试 3 次 → 报警）
- PLC 通讯超时 → 重试 3 次 → 报警
- 图像为空/损坏 → 跳过该帧，记录日志
- 算子执行超时 → 强制终止（CancellationToken）
- 流程文件损坏 → 提示并恢复（备份机制）

### 第 13 周：性能优化
- 图像对象池（Microsoft.Extensions.ObjectPool）
- 采集-处理-输出 流水线（生产者-消费者模式，Channel<T>）
- UI 线程与处理线程分离（Task.Run + Dispatcher.BeginInvoke）
- Halcon 算子并行化（Parallel.ForEach 处理多 ROI）

### 第 14 周：用户操作完善
- 步骤复制/粘贴（支持跨流程）
- 流程模板（预设 3 个常用模板）
- 参数导入/导出（JSON 文件）
- 用户权限（操作员/工程师/管理员，三级权限）
- 操作日志

### 第 15 周：打包 + 文档
- 发布配置（Release 模式）
- 安装器（Wix Toolset 或 ZIP 发布）
- 用户手册（不超过 20 页）
- 维护手册
- 接口文档（DocFX 自动生成）

### 第 16 周：最终测试 + 交付
- 全功能回归测试
- 24 小时连续运行测试
- 修复残余 Bug
- 打 Tag v1.0.0
